setup c programming development environment setup - netbeans IDE (ubuntu 22.04)
==============================================================================
sudo apt install build-essential gcc gdb

sudo apt install default-jdk
sudo apt install default-jre

sudo install netbeans
sudo snap install netbeans --classic

Install c/c++ plugins in apache netbeans : netbeans c/c++ plugin download : n the NetBeans IDE, select Tools > Plugins. In the Available Plugins tab, select the checkbox for Java or C/C++, depending on which is missing from your IDE.
